EQ-28 共包括三个子量表：

1. **认知共情（Cognitive Empathy）**
2. **情绪共情（Emotional Empathy）**
3. **社交技能（Social Skills）**

每个子量表得分可按百分制换算（满分 100），分档描述如下。

---

## 一、认知共情（Cognitive Empathy）

**题目**：1, 11, 14, 15, 22, 26, 29, 34, 35, 36, 38  
**定义**：理解他人心理状态和观点的能力。能够理性地理解他人的意图和想法，而不一定分享其情绪体验。  
**低分特征**：难以换位思考，社交理解能力较弱，可能表现出对他人动机和观点的误解。

| 得分范围 | 档次 | 描述                                                                                                   |
| -------- | ---- | ------------------------------------------------------------------------------------------------------ |
| 0–39     | 低   | 对他人的思维和意图理解能力有限。容易误解他人行为或观点，换位思考困难。需要提升逻辑理解他人意图的能力。 |
| 40–69    | 中   | 能理解他人的心理状态和观点，但在复杂或压力情境下仍可能理解不准确。适合练习多角度思考和理性判断。       |
| 70–100   | 高   | 善于理解他人的思维和意图，能够准确把握他人的观点与心理状态。换位思考能力强，社交理解能力突出。         |

---

## 二、情绪共情（Emotional Empathy）

**题目**：3, 12, 13, 16, 18, 19, 27, 28, 31, 33, 39  
**定义**：感受并共鸣他人情绪的能力，容易吸收他人的情绪。  
**低分特征**：对他人情绪觉察不足，缺乏同理心，难以表现出关心或温暖。

| 得分范围 | 档次 | 描述                                                                                         |
| -------- | ---- | -------------------------------------------------------------------------------------------- |
| 0–39     | 低   | 情绪共鸣能力较弱，对他人情绪感知不足，难以表达同理或关怀。可能在社交关系中显得冷漠或疏离。   |
| 40–69    | 中   | 能感受到他人情绪，但在情绪吸收和共情上存在一定波动。可通过练习情绪识别和情绪表达增强共情力。 |
| 70–100   | 高   | 情绪敏感且易于共情，能够感受到他人情绪并适当回应。善于表达关怀，能够建立温暖的人际关系。     |

---

## 三、社交技能（Social Skills）

**题目**：2, 4, 7, 8, 21, 37  
**定义**：有效沟通、适应社交情境并与他人互动的能力。  
**低分特征**：沟通和社交适应能力弱，可能在社交中出现困惑或不当行为。

| 得分范围 | 档次 | 描述                                                                                                       |
| -------- | ---- | ---------------------------------------------------------------------------------------------------------- |
| 0–39     | 低   | 社交能力有限，沟通不顺畅，难以处理复杂社交情境。需要训练基本社交技巧和互动规则。                           |
| 40–69    | 中   | 社交能力一般，能应对常规人际交往，但在冲突或复杂社交场景下可能表现不佳。可通过练习沟通策略和社交礼仪提高。 |
| 70–100   | 高   | 社交能力强，能够灵活适应不同情境，有效沟通并建立良好的人际关系。能够自如处理冲突与合作。                   |

1. **分档仅供参考**：分数体现 EQ 各子量表能力倾向，不是绝对评判。
2. **可按比例换算**：如果问卷满分不是 100，可按百分比计算：  
   \[
   百分比得分 = \frac{实际得分}{最大得分} \times 100
   \]
3. **反馈应用**：根据各子量表得分，可提供针对性建议：
   - 认知共情低：练习理性换位思考、理解他人观点
   - 情绪共情低：练习情绪识别、情绪表达与关怀
   - 社交技能低：练习沟通技巧、社交适应和互动策略
