# 疯狂动物城人格测评 - 完整题目列表（四维度版本）

## 维度说明

1. **Pace** (行动速度): 慢条斯理(0.0) ↔ 高速执行(1.0)
2. **Orderliness** (秩序偏好): 随性灵活(0.2) ↔ 自律严谨(0.9)
3. **Sincerity** (真诚指数): 策略圆滑(0.3) ↔ 坦率透明(0.85)
4. **Extraversion** (外向呈现): 内向克制(0.1) ↔ 外向张扬(0.95)

## 设计原则总结

1. **维度独立性**: 每个选项可同时影响多个维度,但贡献方向清晰
2. **双向计分**: 既有正向增量(+)也有负向增量(-),精准捕捉对立特质
3. **情境多样性**: 涵盖规则遵守、社交互动、危机应对、人际冲突等多场景
4. **区分度优化**: 不同选项在维度贡献上差异明显(0.05-0.20范围)
5. **角色针对性**: 题目设计直接关联8位角色的典型行为模式
