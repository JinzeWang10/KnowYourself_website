# 🌹《甄嬛传人格测评》设计文档

## 一、测评概念

本测评以《甄嬛传》中的 12 位核心人物为原型，通过现代生活场景题，衡量答题者在情感、理性、策略、自控、进取心等多个心理维度上的特征，判断其最接近哪种"宫廷人格"。

本测评采用**八维度模型**：
- **传统人格维度**（5个）：情感敏感、策略思维、外显能量、理想化、稳定自律
- **现代生活维度**（3个）：进取心、真实性、求新倾向

通过每题选项赋分，计算与 12 个角色的相似度，输出最相似的角色人格结果。

## 二、八大心理维度

### 传统人格维度（5个）
| 维度          | 含义                       |
| ------------- | -------------------------- |
| 🌸 情感敏感度 | 对他人情绪、评价的反应强度 |
| 🧩 策略思维   | 行动前的思考深度、谋略性与控制欲 |
| 💫 外显能量   | 表达力、自信与外向程度     |
| 🪞 情感理想化 | 对爱、道德、理想的执着     |
| 🧱 稳定与自律 | 内在稳定性、责任感与耐心   |

### 现代生活维度（3个）
| 维度          | 含义                       |
| ------------- | -------------------------- |
| 🏃 进取心     | 对成功的渴望、自我驱动力 vs 躺平接受现状 |
| 🎭 真实性     | 表达真我的意愿 vs 社会面具/形象管理 |
| 💡 求新倾向   | 拥抱变化、探索欲望 vs 偏好稳定熟悉 |

## 四、题目与选项映射（28题 - 增量式计分版本）

**题目体系**：
- 共28题，充分覆盖8个维度
- 每个维度在15-20题中有有效贡献（scores ≠ 0）
- 每道题5个选项，每个选项影响1-3个维度（增量式计分）
- 支持正向增量（如 `"energy": 0.18`）和负向增量（如 `"authenticity": -0.12`）

**计分机制变更**（重要）：
- ❌ **旧版**：每个选项对应一个角色，继承该角色的完整8维向量
- ✅ **新版**：每个选项只影响部分维度，使用增量式计分
  - 更灵活：用户人格可以跨角色组合（如70%甄嬛后期+30%皇后）
  - 更精准：负向分数能捕捉对立特质（如高真实性 vs 高伪装）
  - 更公平：分维度统计有效题数，避免稀释分数

**题目示例说明**：下方题目中的"（角色名）"仅作参考，实际计分以代码中的scores字段为准。

## 五、计分与相似度计算规则

### 5.1 基本计算方式（增量式维度贡献模型）

**重要更新**：本测评已从"角色继承模式"升级为"增量式维度贡献模式"，更精准地捕捉用户的多维人格特征。

1. **选项映射**：每个选项对应**增量式维度贡献分数**（而非完整角色向量）
   - 每个选项只影响部分维度，不是全部8个维度
   - 分数可以是正值（增强该维度）、负值（削弱该维度）或0（不影响）
   - 示例：
     ```json
     // Q1选项A："当和事佬，让每个人都感觉被理解"
     {"emotional": 0.16, "idealism": 0.14}
     // 只增强情感敏感和理想化两个维度

     // Q1选项D："表面保持中立，暗中推动自己的方案"
     {"strategy": 0.2, "ambition": 0.16, "authenticity": -0.12}
     // 增强策略思维和进取心，同时降低真实性
     ```

2. **用户向量计算**（改进算法，避免分数稀释）：
   ```
   步骤1：遍历28题，累加每个维度的分数
   步骤2：统计每个维度被累加的有效次数（分数≠0的次数）
   步骤3：每个维度的最终得分 = 该维度累计分数 ÷ 该维度有效累加次数

   关键改进：
   - ❌ 旧算法：所有维度都除以28（总题数），导致分数严重稀释
   - ✅ 新算法：每个维度除以自己的有效题数

   示例：
   - emotional维度在28题中有15题给了分数 → 累计2.5分 ÷ 15 = 0.167 (16.7分/100)
   - strategy维度在28题中有18题给了分数 → 累计3.2分 ÷ 18 = 0.178 (17.8分/100)
   ```

3. **相似度计算**：使用**加权欧式距离**
   ```
   距离 = √(Σ 权重ᵢ × (用户维度ᵢ - 角色维度ᵢ)²)
   相似度 = 1 / (1 + 距离) × 100%
   ```

### 5.2 维度权重设置

为提高区分度，不同维度设置不同权重：

| 维度类别 | 维度名称 | 权重 | 理由 |
|---------|---------|------|------|
| 核心维度 | 策略思维 | 1.3 | 最能区分角色的关键特质 |
| 核心维度 | 进取心 | 1.2 | 现代生活核心分化维度 |
| 核心维度 | 真实性 | 1.2 | 行为模式决定因素 |
| 重要维度 | 情感敏感 | 1.0 | 基础人格特质 |
| 重要维度 | 外显能量 | 1.0 | 社交风格指标 |
| 重要维度 | 求新倾向 | 1.0 | 生活方式选择 |
| 辅助维度 | 理想化 | 0.8 | 与情感敏感相关 |
| 辅助维度 | 稳定自律 | 0.8 | 行为层面特征 |

### 5.3 输出结果

- **主要结果**：相似度最高的角色（相似度 > 60%）
- **次要结果**：如果第二名相似度 > 50%，输出"混合型人格"
  - 例如："你有 65% 的甄嬛后期特质，同时有 52% 的皇后特质"
- **维度雷达图**：展示用户在 8 个维度的得分分布

### 5.4 题目设计原则（增量式模型）

为确保测评准确性：
1. **维度覆盖均衡**：每个维度在20-25题中有贡献，避免某些维度数据不足
2. **增量幅度合理**：单题单维度增量在 0.08-0.20 之间，避免单题主导
3. **正负平衡**：负向分数用于区分极端特质（如真实性 vs 伪装）
4. **情境多样化**：涵盖工作、社交、情感、压力应对等场景
5. **选项差异化**：同一题目的5个选项影响不同的维度组合，提高区分度

**优势**：
- 不再受"角色频次"限制，用户人格特征可以更灵活地跨角色组合
- 负向分数机制能更准确捕捉对立特质（如"真实坦率 vs 形象管理"）
- 分维度统计有效题数，避免分数稀释问题
