# 🌹《甄嬛传人格测评》设计文档

## 一、测评概念

本测评以《甄嬛传》中的 12 位核心人物为原型，通过现代生活场景题，衡量答题者在情感、理性、策略、自控、进取心等多个心理维度上的特征，判断其最接近哪种"宫廷人格"。

本测评采用**八维度模型**：
- **传统人格维度**（5个）：情感敏感、策略思维、外显能量、理想化、稳定自律
- **现代生活维度**（3个）：进取心、真实性、求新倾向

通过每题选项赋分，计算与 12 个角色的相似度，输出最相似的角色人格结果。

---

## 二、八大心理维度

### 传统人格维度（5个）
| 维度          | 含义                       |
| ------------- | -------------------------- |
| 🌸 情感敏感度 | 对他人情绪、评价的反应强度 |
| 🧩 策略思维   | 行动前的思考深度、谋略性与控制欲 |
| 💫 外显能量   | 表达力、自信与外向程度     |
| 🪞 情感理想化 | 对爱、道德、理想的执着     |
| 🧱 稳定与自律 | 内在稳定性、责任感与耐心   |

### 现代生活维度（3个）
| 维度          | 含义                       |
| ------------- | -------------------------- |
| 🏃 进取心     | 对成功的渴望、自我驱动力 vs 躺平接受现状 |
| 🎭 真实性     | 表达真我的意愿 vs 社会面具/形象管理 |
| 💡 求新倾向   | 拥抱变化、探索欲望 vs 偏好稳定熟悉 |

## 四、题目与选项映射（28题 - 8维度版本）

**题目体系**：
- 共28题，充分覆盖8个维度
- 传统人格维度（5个）：每个维度4-7题
- 现代生活维度（3个）：每个维度5-6题

## 五、计分与相似度计算规则

### 5.1 基本计算方式

1. **选项映射**：每个选项对应一个角色，继承该角色的 8 维向量值
   - 例如：选择"甄嬛前期"选项 → 获得向量 [0.8, 0.4, 0.4, 0.9, 0.5, 0.4, 0.7, 0.5]

2. **用户向量计算**：
   ```
   用户在每个维度的得分 = 该维度所有题目得分的平均值
   最终得到用户的 8 维人格向量
   ```

3. **相似度计算**：使用**加权欧式距离**
   ```
   距离 = √(Σ 权重ᵢ × (用户维度ᵢ - 角色维度ᵢ)²)
   相似度 = 1 / (1 + 距离) × 100%
   ```

### 5.2 维度权重设置

为提高区分度，不同维度设置不同权重：

| 维度类别 | 维度名称 | 权重 | 理由 |
|---------|---------|------|------|
| 核心维度 | 策略思维 | 1.3 | 最能区分角色的关键特质 |
| 核心维度 | 进取心 | 1.2 | 现代生活核心分化维度 |
| 核心维度 | 真实性 | 1.2 | 行为模式决定因素 |
| 重要维度 | 情感敏感 | 1.0 | 基础人格特质 |
| 重要维度 | 外显能量 | 1.0 | 社交风格指标 |
| 重要维度 | 求新倾向 | 1.0 | 生活方式选择 |
| 辅助维度 | 理想化 | 0.8 | 与情感敏感相关 |
| 辅助维度 | 稳定自律 | 0.8 | 行为层面特征 |

### 5.3 输出结果

- **主要结果**：相似度最高的角色（相似度 > 60%）
- **次要结果**：如果第二名相似度 > 50%，输出"混合型人格"
  - 例如："你有 65% 的甄嬛后期特质，同时有 52% 的皇后特质"
- **维度雷达图**：展示用户在 8 个维度的得分分布

### 5.4 题目设计原则

为确保测评准确性：
1. **题目平衡**：每个角色出现 4-6 次，避免高频角色偏差
2. **维度覆盖**：每个维度至少 3 道题目测量
3. **情境多样**：涵盖工作、社交、情感、压力应对等场景
4. **选项独立**：同一题目的 5 个选项尽量对应不同角色类型
