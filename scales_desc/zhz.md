# 🌹《甄嬛传人格测评》设计文档

## 一、测评概述

本测评以《甄嬛传》中的 12 位核心人物为原型，通过现代生活场景题，衡量答题者在情感、理性、策略、自控、进取心等多个心理维度上的特征，判断其最接近哪种"宫廷人格"。

### 基本信息
- **题目数量**：30道题（24道强迫选择题 + 6道李克特量表锚点题）
- **测评时长**：8-10分钟
- **维度模型**：六维度双极模型
- **角色数量**：12个经典角色
- **测评类别**：趣味测评

---

## 二、六大心理维度（双极模型）

本测评采用**六维度双极模型**，每个维度都是一个连续谱系，两端代表对立的特质：

| 维度ID | 维度名称 | 双极标签 | 含义说明 |
|--------|---------|---------|---------|
| **sensitivity** | 情绪敏感性 | 稳定冷静 ↔ 敏感细腻 | 对情绪刺激的反应强度 |
| **rationality** | 思维模式 | 直觉冲动 ↔ 理性谋划 | 决策方式与谋略性 |
| **sociability** | 社交能量 | 内向沉静 ↔ 外向张扬 | 社交行为倾向与表达力 |
| **idealism** | 价值取向 | 实用主义 ↔ 理想主义 | 价值权衡倾向与道德感 |
| **ambition** | 野心指数 | 随遇而安 ↔ 进取心强 | 成功追求程度与自我驱动力 |
| **authenticity** | 自我表达 | 战略掩饰 ↔ 真实坦率 | 真实自我展露程度 |

### 维度评分说明
- **0.0-0.3**：低分区间（偏向左极）
- **0.4-0.6**：中等区间（中庸平衡）
- **0.7-1.0**：高分区间（偏向右极）

## 四、题目设计体系

### 4.1 题目类型组成

**第一部分：强迫选择题（24题）**
- 每题5个选项，采用强迫选择机制
- 每个选项对应特定维度的增量分数
- 支持正向增量（如 `"ambition": 0.18`）和负向增量（如 `"authenticity": -0.14`）
- 每个选项影响1-3个维度

**第二部分：锚点量表题（6题）**
- 采用5级李克特量表（1=完全不同意 ~ 5=完全同意）
- 直接测量各维度的绝对水平
- 用于校准强迫选择题的相对分数

### 4.2 题目覆盖主题

题目按主题分为以下几个组别：

| 主题组 | 题目范围 | 主要测量维度 |
|--------|---------|------------|
| **情绪与冲突处理** | Q1-Q4 | 思维模式、社交能量、情绪敏感性 |
| **情感与人际关系** | Q5-Q8 | 价值取向、自我表达、社交能量 |
| **压力与挫折应对** | Q9-Q12 | 情绪敏感性、野心指数、思维模式 |
| **目标与野心** | Q13-Q16 | 野心指数、思维模式、自我表达 |
| **自我认知与价值观** | Q17-Q20 | 价值取向、自我表达、思维模式 |
| **现代生活场景** | Q21-Q24 | 野心指数、自我表达、社交能量 |
| **人格锚点量表** | Q25-Q30 | 六大维度绝对水平测量 |

### 4.3 题目示例

#### 示例1：强迫选择题（Q1）
**题目**：团队开会吵翻天，你的做法是？💼

**选项与维度映射**：
- A. 当和事佬，让每个人都感觉被理解
  `{"sensitivity": 0.18, "idealism": 0.15}`
- B. 先观察局势，找准关键矛盾再行动
  `{"rationality": 0.22, "ambition": 0.12}`
- C. 有话直说，最烦拐弯抹角！
  `{"sociability": 0.22, "authenticity": 0.18}`
- D. 表面保持中立，暗中推动自己的方案
  `{"rationality": 0.24, "ambition": 0.18, "authenticity": -0.14}`

**设计逻辑**：
- 选项D展示了负向计分机制（真实性-0.14），捕捉"战略掩饰"特质
- 选项E通过负向计分体现"低野心"和"情绪稳定"

## 五、计分与相似度计算机制

### 5.1 四阶段向量建模方法

#### **阶段1：维度得分累加（Raw Score Accumulation）**

对于强迫选择题（Q1-Q24）：
```
raw_score[dimension] = Σ option_increment[dimension]
```

每个选项的增量值范围：0.08 ~ 0.24（正向）或 -0.18 ~ -0.08（负向）

#### **阶段2：锚点量表校准（Anchor Calibration）**

对于李克特量表题（Q25-Q30）：
```
anchor_score[dimension] = (user_rating - 1) / 4
```
将1-5分转换为0-1范围的绝对水平分数

#### **阶段3：混合校准（Hybrid Calibration）**

融合强迫选择的相对偏好和锚点量表的绝对水平：
```
calibrated_score[d] = α × relative_score[d] + β × anchor_score[d]
```

其中：
- `α = 0.65`（强迫选择题权重）
- `β = 0.35`（锚点量表权重）
- `relative_score[d] = raw_score[d] / max_possible[d]`（归一化后的相对分数）

**设计意图**：
- 强迫选择题捕捉用户的**相对偏好模式**（更接近哪个角色的决策风格）
- 锚点量表题测量**绝对水平**（避免所有维度都偏低的问题）
- 混合校准平衡两者优势

#### **阶段4：混合相似度匹配（Hybrid Similarity）**

对每个角色计算混合相似度：
```
hybrid_similarity = w1 × cosine_similarity + w2 × (1 - euclidean_distance)
```

其中：
- **余弦相似度**（权重 0.7）：捕捉人格特征模式的方向相似性
- **欧氏距离**（权重 0.3）：补充绝对数值的接近程度
- 最终返回相似度最高的前3个角色

### 5.2 计分机制优势

| 特性 | 传统方法 | 本测评方法 |
|------|---------|-----------|
| **维度独立性** | 零和博弈（某维度高→其他必低） | 维度独立（可同时多个高分） |
| **得分范围** | 易偏低（分散稀释） | 可达70-90+（归一化+校准） |
| **匹配精度** | 单一距离度量 | 混合相似度（余弦+欧氏） |
| **负向特质** | 难以捕捉 | 负向计分捕捉对立特质 |
| **绝对vs相对** | 只测相对偏好 | 混合测量（相对+绝对） |

每个角色包含以下结构化信息：

### 6.1 核心特质（用于分享卡片）

### 6.2 详细特质（用于结果页面）

包含**性格优势**和**潜在风险**两部分，每部分3条要点。

**示例**（甄嬛前期）：

**性格优势**：
1. 情感细腻敏锐：你对情绪的感知力极强，能够深刻理解他人的内心世界，这让你在人际交往中充满同理心和温暖
2. 理想主义情怀：你坚守内心的道德准则和美好信念，对爱情、友情、正义都怀有纯粹的追求，这份赤诚让你散发着独特的魅力
3. 真诚坦率：你不擅长掩饰和算计，待人真诚，这让你容易获得他人的信任，建立深层的情感连接

**潜在风险**：
1. 情感脆弱易受伤：你对情绪刺激的反应强烈，容易因为他人的言行而受到深深的伤害，批评和冷漠都会让你痛苦很久
2. 过度理想化现实：你容易对人对事抱有不切实际的美好期待，当现实与理想差距过大时，会陷入深深的失望和自我怀疑
3. 缺乏自我保护：你不善于用策略和伪装保护自己，在复杂的人际关系中容易被利用或误解，边界感不够清晰可能让你过度付出

## 九、版本历史与优化

### 当前版本特性
- ✅ 六维度双极模型（消除维度重叠）
- ✅ 12个高区分度角色原型
- ✅ 30题混合题型（强迫选择 + 锚点量表）
- ✅ 四阶段向量建模（累加→归一化→校准→匹配）
- ✅ 混合相似度算法（余弦 + 欧氏）
- ✅ 负向计分捕捉对立特质
- ✅ 详细的角色优势与风险分析
- ✅ WebP格式角色图片优化

### 设计原则
1. **维度互斥性**：各维度独立，避免概念重叠
2. **角色区分度**：每个角色至少有3个极端维度（≥0.8或≤0.3）
3. **题目均衡性**：每个维度在15-20题中有贡献
4. **计分科学性**：混合相对偏好和绝对水平
5. **结果解释性**：提供优势、风险和行动建议
