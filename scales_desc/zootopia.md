# 🐾《疯狂动物城人格测评》设计文档

## 一、测评概述

本测评以《疯狂动物城》中的 8 位核心角色为原型，通过现代生活场景题，衡量答题者在信任、规则、社交、行动力、理想等多个心理维度上的特征，判断其最接近哪种"动物城人格"。

### 基本信息

- **题目数量**：25 道题（20 道强迫选择题 + 5 道李克特量表锚点题）
- **测评时长**：6-8 分钟
- **维度模型**：四维度双极模型
- **角色数量**：8 个经典角色
- **测评类别**：趣味测评

---

## 二、四大心理维度

一、四大核心维度

1. Pace — 行动速度（慢条斯理 ↔ 高速执行）

反映角色处理任务的速度、反应节奏、行动倾向。

低分：极慢、谨慎、步调松弛（典型：闪电）。

高分：迅速果断、执行效率极高（典型：朱迪）。

2. Orderliness — 秩序偏好（随性灵活 ↔ 自律严谨）

衡量个人对规则、规范、流程的接受度与依赖程度。

低分：随性、不按常规、灵活变通（尼克、Mr. Big）。

高分：严格、自律、注重制度与流程（朱迪、Lionheart、Clawhauser）。

3. Sincerity — 真诚指数（策略圆滑 ↔ 坦率透明）

反映角色的动机透明度、表达方式、言行一致程度。

低分：隐藏动机、有策略、有心机（夏奇羊、尼克）。

高分：直率、真诚、情绪外显（朱迪、闪电）。

4. Extraversion — 外向呈现（内向克制 ↔ 外向张扬）

衡量角色在社交场合中的表达方式、存在感与能量来源。

低分：安静、低调、社交能量消耗（闪电、Mr. Big）。

高分：主动表达、热情、活跃（Clawhauser、朱迪、Gary）。

## 四、角色详细特质描述

1. 🐰 朱迪 Judy Hopps – 理想驱动的高速行动派

核心特质（100–150 字）：
你像朱迪一样，内心怀着强烈的理想主义，相信只要努力便能突破环境限制、实现自我价值。你行动迅速，执行力强，对规则和原则保持高度尊重，同时又拥有积极的社交能量，让你在团队中总是格外亮眼。你乐观、真诚、朝气十足，是那种能点亮现场氛围、也能推动团队前进的人。你对公平和正义有天然敏感度，更愿意主动改变不合理的现状，而不是被动接受。

性格优势：

行动迅捷果断：你不拖延、不犹豫，总能第一时间把计划落实，让进度保持高速推进。

价值观稳定真诚：你坚持原则，坦率表达，是别人眼中可靠、可信任的伙伴。

高度社交亲和力：你善于沟通、热情积极，能快速建立关系并激励周围的人。

潜在风险：

容易过度投入：你的理想主义可能让你在复杂现实前感到挫败，需要适度留有弹性。

速度过快忽略细节：快节奏常让你遗漏风险点，应偶尔放慢步伐审视全局。

对人信任度较高：当遇到心机较深的人，你可能需要更谨慎地辨别动机。

2. 🦊 尼克 Nick Wilde – 机智灵活的游刃派谈判者

核心特质（100–150 字）：
你像尼克一样拥有敏锐的洞察力、超强的情境判断与灵活变通能力。你不按常规出牌，懂得根据局势调整策略，是天生的“社会导航高手”。你擅长观察人性，也擅长用幽默化解尴尬，让你在人际场合总能保持游刃有余。你不喜欢僵硬的规则，更倾向以智慧和机敏突破障碍。虽然你看似随性，但骨子里却有自己坚持的价值线，只是选择更隐蔽、更柔软的表达方式。

性格优势：

策略能力出众：你善于看清局势、灵活应对，总能找到聪明的解决方案。

表达风趣自然：你让人如沐春风，是人际沟通中的调节者和“润滑剂”。

适应力极强：面对变化你毫不畏惧，能轻松切换节奏与角色。

潜在风险：

动机易被误解：你的策略性有时让人难以看透，可能被误判为“不够真诚”。

对规则兴趣不高：可能在需要结构化、高纪律的环境中感到束缚。

容易隐藏真实感受：过度依赖幽默与机智，可能掩盖需要被正视的情绪。

3. 🦁 马市长 Lionheart – 威仪十足的秩序守护者

核心特质（100–150 字）：
你像马市长一样稳重、有权威感，关注整体秩序与结构性的长远发展。你擅长规划，也愿意承担责任，是那种在压力时刻依旧能稳住情绪、带领团队的人。你对规则与制度保持尊重，同时具备一定的理想主义，希望通过组织、制度与领导力推动世界更好地运作。你的外向气场适中，不张扬却能让人感到可靠，是团队中的定心丸。

性格优势：

组织与规划能力强：你能统筹资源、洞察全局，擅长做长期布局。

稳定的领导气场：你冷静、有担当，关键时刻能给人安全感。

原则感明确：你坚持秩序与责任，是团队中值得信赖的中坚力量。

潜在风险：

容易承担过多责任：你可能无意识承担超出自己负荷的事务。

过度强调秩序：在高度灵活的情境中可能显得不够适应。

情绪表达较克制：可能让他人误以为你距离感较强。

4. 🐆 雪豹警官 Clawhauser – 热情外向的温暖社交家

核心特质（100–150 字）：
你像雪豹警官一样亲切、热情、真诚，是社交场合中能迅速让氛围变轻松的人。你的外向特质明显，发自内心喜欢帮助他人，为团队带来积极能量。虽然你对规则保持尊重，但你并非僵硬执行者，而是带着温度的秩序维持者。你情绪透明、表达自然，不会刻意隐藏自己的喜好，是那种让人放下戒心、快速建立信任的伙伴。

性格优势：

极高的亲和力：你给人安全、温暖、愉快的感受，人际关系轻松顺畅。

真诚而坦率：你几乎不掩饰情绪，让人感到可信、可依赖。

乐于支持他人：你愿意贡献情绪能量，是团队气氛的核心来源之一。

潜在风险：

容易过度付出：你为了照顾关系可能牺牲自己的需求。

情绪外显度高：在严肃或高压场合可能被认为“过于投入”。

对冲突较敏感：你可能会回避直接对抗，以避免破坏和谐。

5. 🦥 闪电 Flash – 敏感温和的慢节奏观察者

核心特质（100–150 字）：
你像闪电一样步调沉稳、动作缓慢、不被外界急躁节奏影响。你做事谨慎、细致、重视体验胜过速度。你真诚、坦率，很少玩心机；你的表达常常温和，不争不抢，却拥有独特的洞察力——因为你看得比别人更细、更深。你偏内向，享受独处与稳定的环境，在熟悉的人面前却能展现幽默的一面，是人群中“安静但可信赖”的存在。

性格优势：

极高耐心：你能稳定处理长期任务，不被外界干扰节奏。

真诚无负担：你动机透明、表达简单直接，是让人安心的类型。

深度观察者：你能捕捉细节，常在关键时刻提出关键见解。

潜在风险：

节奏与外界不匹配：在要求快速回应的环境中可能感到压力。

表达过于克制：可能让他人无法准确理解你的真实想法。

容易被忽视：你不主动争取机会，可能错过应得的资源。

6. 🐭 大先生 Mr. Big – 沉静稳重的战略掌控者

核心特质（100–150 字）：
你像大先生一样低调、冷静、重视掌控节奏，不会被情绪左右。你做事谨慎精准，即便行动不快，却能确保每一步都踏稳。你不喜欢引人注目，但你具备过人的判断力与战略思维，会在最关键的时刻作出决定。你社交上偏内敛，不喜欢流于表面的互动，更注重忠诚、信任与长久关系。你少言，但每句话都带分量。

性格优势：

稳定且果断：你做决定不多，但一旦做出，便坚定执行。

高度策略性：你擅长布局，能在复杂局势中保持冷静。

社交忠诚度高：你珍惜信任关系，是值得托付的存在。

潜在风险：

情绪表达不足：你的冷静可能让他人误以为你疏远或冷漠。

社交圈狭窄：你不主动扩展关系，容易形成信息闭环。

过度谨慎：可能在关键机会前犹豫太久。

7. 🐍 盖瑞 Gary – 灵活稳健的情境型执行者

核心特质（100–150 字）：
你像盖瑞一样执行力强、适应性佳，能在不同角色之间切换自如。你既不像高速行动派那样冲动，也不像慢节奏者那样迟缓，而是用“恰到好处”的速度完成任务。你对规则保持适度尊重，但不会被其限制；你真诚但不失分寸，社交能量稳定又让人感到舒服。你是团队中“可靠又不打扰”的存在，擅长用稳健方式推进目标。

性格优势：

节奏平衡良好：你能根据环境调整行动速度，保持效率与稳妥兼具。

情绪稳定：你不易被外界影响，面对压力能维持冷静。

社交自然温和：你不会刻意讨好，却容易让人产生好感。

潜在风险：

容易被低估：你不张扬，可能让别人忽略你的能力与贡献。

角色切换过多：在多任务环境中可能一度感到能量被分散。

对原则界线不够鲜明：你倾向顾及他人，可能在关键冲突中犹豫。

8. 🐏 Gazelle – 魅力四射的公众偶像

核心特质（100–150 字）：
你像 Gazelle 一样拥有强烈的舞台魅力和公众影响力，优雅自信，擅长用艺术与情感连接他人。你兼具亲和力与专业性，既能在聚光灯下展现光彩，也能以柔和的方式影响周围人。你重视信念与表达，不喜欢被条条框框束缚，善于运用个人魅力推动社会话题或情感共鸣。你是那种能用温度和力量同时打动别人的人。

性格优势：

强大号召力：你能迅速吸引关注并影响群体情绪或观点。

情感表达丰富：你擅长以艺术化的方式沟通，容易触动他人的内心。

适应公众场域：你在社交与公开场合中游刃有余，能有效把握场面氛围。

潜在风险：

过度在意他人期待：公众形象压力可能让你忽视自我需求与界限。

对规则反感：你追求表达自由，遇到严格流程或官僚时可能感到受限或抵触。

情绪曝光带来脆弱：高度情感化的表达有时会被放大，带来外界评论或误读。

## 五、题目设计体系

### 5.1 题目类型组成

**第一部分：强迫选择题（20 题）**

- 每题 5 个选项，采用强迫选择机制
- 每个选项对应特定维度的增量分数
- 支持正向增量（如 `"trust": 0.15`）和负向增量（如 `"trust": -0.08`）
- 每个选项影响 1-3 个维度

**第二部分：锚点量表题（5 题）**

- 采用 5 级李克特量表（1=完全不同意 ~ 5=完全同意）
- 直接测量各维度的绝对水平
- 用于校准强迫选择题的相对分数

### 5.2 题目覆盖主题

题目按主题分为以下几个组别：

| 主题组             | 题目范围 | 主要测量维度                 |
| ------------------ | -------- | ---------------------------- |
| **规则与秩序场景** | Q1-Q4    | 秩序偏好、真诚指数、行动速度 |
| **决策与行动应对** | Q5-Q8    | 行动速度、秩序偏好、真诚指数 |
| **冲突与人际处理** | Q9-Q12   | 外向呈现、真诚指数、秩序偏好 |
| **工作与协作场景** | Q13-Q16  | 行动速度、秩序偏好、外向呈现 |
| **价值观与选择**   | Q17-Q20  | 真诚指数、行动速度、外向呈现 |
| **人格锚点量表**   | Q21-Q25  | 四大维度绝对水平测量         |

### 5.3 题目示例

#### 示例 1：强迫选择题（Q1）

**题目**：作为疯狂动物城的市民，你在街上散步时看到朱迪正劝说一只乱扔垃圾的小动物，你会怎么做？

**选项与维度映射**：

- A. 主动上前支持朱迪，一起劝说
  `{"pace": 0.12, "orderliness": 0.15, "sincerity": 0.10, "extraversion": 0.12}`
- B. 观察情况，不插手
  `{"pace": -0.10, "extraversion": -0.08}`
- C. 拍照发社交平台提醒大家
  `{"pace": 0.08, "orderliness": 0.05, "extraversion": 0.15}`
- D. 偷偷把垃圾捡起来，但不说话
  `{"pace": 0.05, "orderliness": 0.12, "sincerity": 0.08, "extraversion": -0.10}`
- E. 笑着离开，不理会
  `{"pace": -0.05, "orderliness": -0.12, "sincerity": -0.08}`

**设计逻辑**：

- 选项 B 和 E 展示了负向计分机制，捕捉"低行动速度"和"低秩序偏好"特质
- 选项 A 同时影响四个维度，全面测量积极行动特质

**选项**：1=完全不同意 → 5=完全同意

**维度映射**：直接测量 `orderliness` 维度的绝对水平

## 六、计分与相似度计算机制

### 6.1 四阶段向量建模方法

#### **阶段 1：维度得分累加（Raw Score Accumulation）**

对于强迫选择题（Q1-Q20）：

```
raw_score[dimension] = Σ option_increment[dimension]
```

每个选项的增量值范围：0.05 ~ 0.18（正向）或 -0.12 ~ -0.05（负向）

#### **阶段 2：锚点量表校准（Anchor Calibration）**

对于李克特量表题（Q21-Q25）：

```
anchor_score[dimension] = (user_rating - 1) / 4
```

将 1-5 分转换为 0-1 范围的绝对水平分数

#### **阶段 3：混合校准（Hybrid Calibration）**

融合强迫选择的相对偏好和锚点量表的绝对水平：

```
calibrated_score[d] = α × relative_score[d] + β × anchor_score[d]
```

其中：

- `α = 0.65`（强迫选择题权重）
- `β = 0.35`（锚点量表权重）
- `relative_score[d] = raw_score[d] / max_possible[d]`（归一化后的相对分数）

**设计意图**：

- 强迫选择题捕捉用户的**相对偏好模式**（更接近哪个角色的决策风格）
- 锚点量表题测量**绝对水平**（避免所有维度都偏低的问题）
- 混合校准平衡两者优势

#### **阶段 4：混合相似度匹配（Hybrid Similarity）**

对每个角色计算混合相似度：

```
hybrid_similarity = w1 × cosine_similarity + w2 × (1 - euclidean_distance)
```

其中：

- **余弦相似度**（权重 0.7）：捕捉人格特征模式的方向相似性
- **欧氏距离**（权重 0.3）：补充绝对数值的接近程度
- 最终返回相似度最高的前 3 个角色

### 6.2 计分机制优势

| 特性             | 传统方法                        | 本测评方法                 |
| ---------------- | ------------------------------- | -------------------------- |
| **维度独立性**   | 零和博弈（某维度高 → 其他必低） | 维度独立（可同时多个高分） |
| **得分范围**     | 易偏低（分散稀释）              | 可达 70-90+（归一化+校准） |
| **匹配精度**     | 单一距离度量                    | 混合相似度（余弦+欧氏）    |
| **负向特质**     | 难以捕捉                        | 负向计分捕捉对立特质       |
| **绝对 vs 相对** | 只测相对偏好                    | 混合测量（相对+绝对）      |

## 十、版本历史与设计原则

### 当前版本特性

- ✅ 四维度双极模型（消除维度重叠）
- ✅ 8 个高区分度角色原型
- ✅ 25 题混合题型（强迫选择 + 锚点量表）
- ✅ 四阶段向量建模（累加 → 归一化 → 校准 → 匹配）
- ✅ 混合相似度算法（余弦 + 欧氏）
- ✅ 负向计分捕捉对立特质
- ✅ 详细的角色优势与风险分析
- ✅ WebP 格式角色图片优化

### 设计原则

1. **维度互斥性**：各维度独立，避免概念重叠
2. **角色区分度**：每个角色至少有 2 个极端维度（≥0.8 或 ≤0.3）
3. **题目均衡性**：每个维度在 12-15 题中有贡献
4. **计分科学性**：混合相对偏好和绝对水平
5. **结果解释性**：提供优势、风险和行动建议
