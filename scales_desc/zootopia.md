# 🐾《疯狂动物城人格测评》设计文档

## 一、测评概述

本测评以《疯狂动物城》中的 8 位核心角色为原型，通过现代生活场景题，衡量答题者在信任、规则、社交、行动力、理想等多个心理维度上的特征，判断其最接近哪种"动物城人格"。

### 基本信息

- **题目数量**：25 道题（20 道强迫选择题 + 5 道李克特量表锚点题）
- **测评时长**：6-8 分钟
- **维度模型**：五维度双极模型
- **角色数量**：8 个经典角色
- **测评类别**：趣味测评

---

## 二、五大心理维度（双极模型）

本测评采用**五维度双极模型**，每个维度都是一个连续谱系，两端代表对立的特质：

| 维度 ID          | 维度名称     | 双极标签            | 含义说明                     |
| ---------------- | ------------ | ------------------- | ---------------------------- |
| **trust**        | 社会信任度   | 警惕怀疑 ↔ 信任开放 | 对他人和社会的信任与合作意愿 |
| **lawfulness**   | 律法遵从度   | 灵活变通 ↔ 严守规则 | 遵守规则、制度的倾向         |
| **pace**         | 行动节奏     | 从容慢调 ↔ 快速果断 | 做事快慢、果断与从容         |
| **extraversion** | 社交呈现方式 | 内敛低调 ↔ 外向张扬 | 社交行为倾向与表达力         |
| **idealism**     | 理想主义指数 | 实用主义 ↔ 理想主义 | 重视原则与理想的程度         |

### 维度评分说明

- **0.0-0.3**：低分区间（偏向左极）
- **0.4-0.6**：中等区间（中庸平衡）
- **0.7-1.0**：高分区间（偏向右极）

## 四、角色详细特质描述

### 1. 🐰 朱迪 Judy Hopps - 理想主义的行动派警官

**核心特质**（用于分享卡片，100-150 字）：
你像朱迪一样充满理想和正义感，相信每个人都能通过努力改变世界。你信任他人，严格遵守规则，行动力超强，社交能力出色。你勇于挑战不可能，为了目标全力以赴。你相信公平与正义，愿意为理想付出一切努力。

**性格优势**：

1. **超强行动力**：你像朱迪一样，一旦确定目标就会立即行动，不畏艰难，勇于尝试新事物，这让你在工作和生活中充满冲劲和活力
2. **高度信任与合作**：你对他人抱有信任，乐于与人合作，这让你容易建立良好的人际关系，获得他人的支持和帮助
3. **理想主义驱动**：你追求公平与正义，有强烈的道德感和使命感，这份理想主义让你在困境中依然坚持初心，成为他人的榜样

**潜在风险**：

1. **过度理想化**：你容易对人对事抱有过高期待，当现实与理想差距过大时，会感到深深的失望和挫败，需要学会接受不完美
2. **冲动决策风险**：你的高行动力有时会让你在未充分思考时就做出决定，可能忽略细节或潜在风险，导致不必要的麻烦
3. **易被利用**：你的高信任度让你容易相信他人，在复杂的人际关系中可能被狡猾的人利用，需要提升警觉性和边界感

---

### 2. 🦊 尼克 Nick Wilde - 机智灵活的现实主义者

**核心特质**：
你像尼克一样聪明狡黠、机智灵活，善于在复杂环境中找到生存之道。你对人保持适度的警惕，不会轻易信任他人，但内心依然保留一份善良。你务实而灵活，行动力强，能快速适应变化。你懂得权衡利弊，用理性而非情感做决策。

**性格优势**：

1. **机智应变能力**：你善于观察局势，快速找到问题的解决方案，在复杂的人际关系和工作环境中游刃有余
2. **社交灵活性**：你懂得根据不同情况调整自己的表达方式，能与各种类型的人相处，这让你在职场和生活中如鱼得水
3. **现实主义智慧**：你不会被理想蒙蔽双眼，能够看清现实，做出务实的选择，这让你避免了很多不必要的风险

**潜在风险**：

1. **信任障碍**：你对他人的信任度较低，可能因过度警惕而错失真诚的友谊和合作机会，长期来看可能让你感到孤独
2. **理想主义欠缺**：你过于务实可能让你失去追求更高目标的动力，有时会在"够用就好"的心态中停滞不前
3. **行动风险**：你的快速决策和行动力虽然有效，但有时可能因为过于追求效率而忽略长期影响

---

### 3. 🦁 马市长 Mayor Lionheart - 守序理性的城市管理者

**核心特质**：
你像马市长一样守规矩、理性冷静，做事有条不紊。你高度遵守规则和制度，以秩序和稳定为优先，处理事务沉稳而系统。你对他人保持适度信任，能理性判断形势。你不追求个人风头，更注重实际效果和长远利益。

**性格优势**：

1. **高度守规有序**：你严格遵守规则和制度，做事有条不紊，这让你在组织管理和执行任务时表现出色，值得信赖
2. **冷静理性决策**：你善于统筹全局，能够在复杂情况下保持冷静，做出理性的判断和决策，避免情绪化的失误
3. **适度信任平衡**：你对他人的信任度适中，既不过度怀疑也不盲目相信，能够合理评估风险和机会

**潜在风险**：

1. **行动节奏慢**：你过于注重计划和稳妥，可能在需要快速决策时错失良机，在快节奏的环境中可能显得反应迟缓
2. **灵活性不足**：你过于依赖规则和既定流程，面对突发情况或创新需求时可能缺乏应变能力
3. **社交亲近感弱**：你的中庸社交风格虽然不会得罪人，但也难以建立深层的情感连接，可能让你在团队中显得疏离

### 4. 🐆 雪豹警官 Clawhauser - 职业忠诚的稳健执行者

**核心特质**：
你像雪豹警官一样职业忠诚、稳重果敢，有强烈的职业原则和执行力。你严格遵守规则，情绪稳定，适合处理需要精准执行的任务。你内敛低调，不喜欢过多的社交，更倾向于用行动而非言语表达。你务实而谨慎，对人保持一定距离。

**性格优势**：

1. **高度职业忠诚**：你秉公执法、严格遵守职业规范，这让你在需要原则和纪律的环境中表现出色，赢得他人尊重
2. **情绪稳定可靠**：你情绪稳定，不易受外界干扰，适合处理紧急事件和高压任务，是团队中的稳定力量
3. **稳步推进能力**：你行动节奏适中，能够稳步推进任务，不急不躁，确保每一步都扎实可靠

**潜在风险**：

1. **社交能力受限**：你的内敛低调可能让你在需要沟通协调的场合表现不佳，难以建立广泛的人际网络
2. **创新能力欠缺**：你的理想主义不足，可能缺乏对新想法和创新的追求，容易陷入"按部就班"的工作模式
3. **信任度低**：你对他人的信任度较低，可能造成团队协作中的隔阂，影响团队凝聚力

## 五、题目设计体系

### 5.1 题目类型组成

**第一部分：强迫选择题（20 题）**

- 每题 5 个选项，采用强迫选择机制
- 每个选项对应特定维度的增量分数
- 支持正向增量（如 `"trust": 0.15`）和负向增量（如 `"trust": -0.08`）
- 每个选项影响 1-3 个维度

**第二部分：锚点量表题（5 题）**

- 采用 5 级李克特量表（1=完全不同意 ~ 5=完全同意）
- 直接测量各维度的绝对水平
- 用于校准强迫选择题的相对分数

### 5.2 题目覆盖主题

题目按主题分为以下几个组别：

| 主题组             | 题目范围 | 主要测量维度             |
| ------------------ | -------- | ------------------------ |
| **规则与信任场景** | Q1-Q4    | 律法遵从度、社会信任度   |
| **决策与风险应对** | Q5-Q8    | 行动节奏、理想主义指数   |
| **冲突与人际处理** | Q9-Q12   | 社交呈现方式、社会信任度 |
| **工作与协作场景** | Q13-Q16  | 律法遵从度、行动节奏     |
| **价值观与选择**   | Q17-Q20  | 理想主义指数、社会信任度 |
| **人格锚点量表**   | Q21-Q25  | 五大维度绝对水平测量     |

### 5.3 题目示例

#### 示例 1：强迫选择题（Q1）

**题目**：作为疯狂动物城的市民，你在街上散步时看到朱迪正劝说一只乱扔垃圾的小动物，你会怎么做？

**选项与维度映射**：

- A. 主动上前支持朱迪，一起劝说
  `{"lawfulness": 0.15, "trust": 0.10}`
- B. 观察情况，不插手
  `{"pace": -0.05}`
- C. 拍照发社交平台提醒大家
  `{"extraversion": 0.12, "idealism": 0.05}`
- D. 偷偷把垃圾捡起来，但不说话
  `{"lawfulness": 0.10}`
- E. 笑着离开，不理会
  `{"trust": -0.05}`

**设计逻辑**：

- 选项 B 和 E 展示了负向计分机制，捕捉"低行动力"和"低信任"特质
- 选项 A 同时影响两个维度（规则意识+信任）

#### 示例 2：锚点量表题（Q21）

**题目**：我倾向于严格遵守规则和制度

**选项**：1=完全不同意 → 5=完全同意

**维度映射**：直接测量 `lawfulness` 维度的绝对水平

## 六、计分与相似度计算机制

### 6.1 四阶段向量建模方法

#### **阶段 1：维度得分累加（Raw Score Accumulation）**

对于强迫选择题（Q1-Q20）：

```
raw_score[dimension] = Σ option_increment[dimension]
```

每个选项的增量值范围：0.05 ~ 0.18（正向）或 -0.12 ~ -0.05（负向）

#### **阶段 2：锚点量表校准（Anchor Calibration）**

对于李克特量表题（Q21-Q25）：

```
anchor_score[dimension] = (user_rating - 1) / 4
```

将 1-5 分转换为 0-1 范围的绝对水平分数

#### **阶段 3：混合校准（Hybrid Calibration）**

融合强迫选择的相对偏好和锚点量表的绝对水平：

```
calibrated_score[d] = α × relative_score[d] + β × anchor_score[d]
```

其中：

- `α = 0.65`（强迫选择题权重）
- `β = 0.35`（锚点量表权重）
- `relative_score[d] = raw_score[d] / max_possible[d]`（归一化后的相对分数）

**设计意图**：

- 强迫选择题捕捉用户的**相对偏好模式**（更接近哪个角色的决策风格）
- 锚点量表题测量**绝对水平**（避免所有维度都偏低的问题）
- 混合校准平衡两者优势

#### **阶段 4：混合相似度匹配（Hybrid Similarity）**

对每个角色计算混合相似度：

```
hybrid_similarity = w1 × cosine_similarity + w2 × (1 - euclidean_distance)
```

其中：

- **余弦相似度**（权重 0.7）：捕捉人格特征模式的方向相似性
- **欧氏距离**（权重 0.3）：补充绝对数值的接近程度
- 最终返回相似度最高的前 3 个角色

### 6.2 计分机制优势

| 特性             | 传统方法                        | 本测评方法                 |
| ---------------- | ------------------------------- | -------------------------- |
| **维度独立性**   | 零和博弈（某维度高 → 其他必低） | 维度独立（可同时多个高分） |
| **得分范围**     | 易偏低（分散稀释）              | 可达 70-90+（归一化+校准） |
| **匹配精度**     | 单一距离度量                    | 混合相似度（余弦+欧氏）    |
| **负向特质**     | 难以捕捉                        | 负向计分捕捉对立特质       |
| **绝对 vs 相对** | 只测相对偏好                    | 混合测量（相对+绝对）      |
