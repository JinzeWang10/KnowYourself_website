# 🐾《疯狂动物城人格测评》设计文档

## 一、测评概述

本测评以《疯狂动物城》中的 8 位核心角色为原型，通过现代生活场景题，衡量答题者在信任、规则、社交、行动力、理想等多个心理维度上的特征，判断其最接近哪种"动物城人格"。

### 基本信息

- **题目数量**：25 道题（20 道强迫选择题 + 5 道李克特量表锚点题）
- **测评时长**：6-8 分钟
- **维度模型**：五维度双极模型
- **角色数量**：8 个经典角色
- **测评类别**：趣味测评

---

## 二、四大心理维度

一、四大核心维度

1. Pace — 行动速度（慢条斯理 ↔ 高速执行）

反映角色处理任务的速度、反应节奏、行动倾向。

低分：极慢、谨慎、步调松弛（典型：闪电）。

高分：迅速果断、执行效率极高（典型：朱迪）。

2. Orderliness — 秩序偏好（随性灵活 ↔ 自律严谨）

衡量个人对规则、规范、流程的接受度与依赖程度。

低分：随性、不按常规、灵活变通（尼克、Mr. Big）。

高分：严格、自律、注重制度与流程（朱迪、Winddancer、Clawhauser）。

3. Sincerity — 真诚指数（策略圆滑 ↔ 坦率透明）

反映角色的动机透明度、表达方式、言行一致程度。

低分：隐藏动机、有策略、有心机（夏奇羊、尼克）。

高分：直率、真诚、情绪外显（朱迪、闪电）。

4. Extraversion — 外向呈现（内向克制 ↔ 外向张扬）

衡量角色在社交场合中的表达方式、存在感与能量来源。

低分：安静、低调、社交能量消耗（闪电、Mr. Big）。

高分：主动表达、热情、活跃（Clawhauser、朱迪、Gary）。

5. Ambition — 进取心（安于现状 ↔ 强烈目标驱动） ← 新增

衡量个人的成就动机、职业/人生的主动追求程度及奋斗心态。

低分：追求稳定、安全、舒适、少欲望（闪电、Mr. Big）。

高分：有目标、有野心、有推动力，愿意主动争取机会（朱迪、夏奇羊、马市长）。

### 5.1 题目类型组成

**第一部分：强迫选择题（20 题）**

- 每题 5 个选项，采用强迫选择机制

- 支持正向增量（如 `"trust": 0.15`）和负向增量（如 `"trust": -0.08`）

- 采用 5 级李克特量表（1=完全不同意 ~ 5=完全同意）
- 直接测量各维度的绝对水平
- 用于校准强迫选择题的相对分数

## 六、计分与相似度计算机制

### 6.1 四阶段向量建模方法

#### **阶段 1：维度得分累加（Raw Score Accumulation）**

对于强迫选择题（Q1-Q20）：

```
raw_score[dimension] = Σ option_increment[dimension]
```

每个选项的增量值范围：0.05 ~ 0.18（正向）或 -0.12 ~ -0.05（负向）

#### **阶段 2：锚点量表校准（Anchor Calibration）**

对于李克特量表题（Q21-Q25）：

```
anchor_score[dimension] = (user_rating - 1) / 4
```

将 1-5 分转换为 0-1 范围的绝对水平分数

#### **阶段 3：混合校准（Hybrid Calibration）**

融合强迫选择的相对偏好和锚点量表的绝对水平：

```
calibrated_score[d] = α × relative_score[d] + β × anchor_score[d]
```

其中：

- `α = 0.65`（强迫选择题权重）
- `β = 0.35`（锚点量表权重）
- `relative_score[d] = raw_score[d] / max_possible[d]`（归一化后的相对分数）

**设计意图**：

- 强迫选择题捕捉用户的**相对偏好模式**（更接近哪个角色的决策风格）
- 锚点量表题测量**绝对水平**（避免所有维度都偏低的问题）
- 混合校准平衡两者优势

#### **阶段 4：混合相似度匹配（Hybrid Similarity）**

对每个角色计算混合相似度：

```
hybrid_similarity = w1 × cosine_similarity + w2 × (1 - euclidean_distance)
```

其中：

- **余弦相似度**（权重 0.7）：捕捉人格特征模式的方向相似性
- **欧氏距离**（权重 0.3）：补充绝对数值的接近程度
- 最终返回相似度最高的前 3 个角色

### 6.2 计分机制优势

| 特性             | 传统方法                        | 本测评方法                 |
| ---------------- | ------------------------------- | -------------------------- |
| **维度独立性**   | 零和博弈（某维度高 → 其他必低） | 维度独立（可同时多个高分） |
| **得分范围**     | 易偏低（分散稀释）              | 可达 70-90+（归一化+校准） |
| **匹配精度**     | 单一距离度量                    | 混合相似度（余弦+欧氏）    |
| **负向特质**     | 难以捕捉                        | 负向计分捕捉对立特质       |
| **绝对 vs 相对** | 只测相对偏好                    | 混合测量（相对+绝对）      |

## 十、版本历史与设计原则

### 当前版本特性

- ✅ 四维度双极模型（消除维度重叠）
- ✅ 8 个高区分度角色原型
- ✅ 25 题混合题型（强迫选择 + 锚点量表）
- ✅ 四阶段向量建模（累加 → 归一化 → 校准 → 匹配）
- ✅ 混合相似度算法（余弦 + 欧氏）
- ✅ 负向计分捕捉对立特质
- ✅ 详细的角色优势与风险分析
- ✅ WebP 格式角色图片优化

### 设计原则

1. **维度互斥性**：各维度独立，避免概念重叠
2. **角色区分度**：每个角色至少有 2 个极端维度（≥0.8 或 ≤0.3）
3. **题目均衡性**：每个维度在 12-15 题中有贡献
4. **计分科学性**：混合相对偏好和绝对水平
5. **结果解释性**：提供优势、风险和行动建议
