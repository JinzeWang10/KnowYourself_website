# 《甄嬛传人格测评》八维模型升级说明

## 一、升级原因

### 原6维模型的问题
1. **维度重合**：
   - "策略心机" 和 "支配控制" 高度相关（相关系数约0.85）
   - 皇后、甄嬛后期、宁贵人这三个角色在两个维度上几乎同步

2. **缺乏现代性**：
   - 维度主要反映传统人格理论（情绪、认知、行为）
   - 未涵盖当代年轻人关注的核心议题：内卷vs躺平、真实vs人设、稳定vs折腾

3. **区分度不足**：
   - 某些角色的维度轮廓过于相似
   - 测评容易偏向高频出现的角色

---

## 二、八维模型结构

### 传统人格维度（5个）

#### 1. 🌸 情感敏感度
- **测量**：对他人情绪、评价的反应强度
- **保留理由**：人际关系的基础特质
- **高分角色**：安陵容、温太医、甄嬛前期

#### 2. 🧩 策略思维（整合原"策略心机"+"支配控制"）
- **测量**：行动前的思考深度、谋略性、控制环境的倾向
- **整合理由**：策略和控制是一体两面，统一为"战略性思维"
- **高分角色**：皇后、甄嬛后期、宁贵人、皇上

#### 3. 💫 外显能量
- **测量**：表达力、自信与外向程度
- **保留理由**：社交风格的核心维度
- **高分角色**：华妃、果郡王、苏培盛

#### 4. 🪞 情感理想化
- **测量**：对爱、道德、理想的执着
- **保留理由**：价值观维度，影响重大决策
- **高分角色**：甄嬛前期、果郡王、温太医

#### 5. 🧱 稳定与自律
- **测量**：内在稳定性、责任感与耐心
- **保留理由**：行为执行力指标
- **高分角色**：沈眉庄、皇后

---

### 现代生活维度（3个）

#### 6. 🏃 进取心
- **测量**：对成功的渴望、自我驱动力 vs 躺平接受现状
- **新增理由**：Z世代最核心的人格分化维度
- **应用场景**：
  - 职业选择（内卷 vs 躺平）
  - 生活态度（上进 vs 佛系）
  - 竞争意识

**高分角色**：皇后(0.9)、皇上(0.9)、甄嬛后期(0.85)、浣碧(0.8)
**低分角色**：温太医(0.2)、苏培盛(0.4)

#### 7. 🎭 真实性
- **测量**：表达真我的意愿 vs 社会面具/形象管理
- **新增理由**：社交媒体时代的核心议题
- **应用场景**：
  - 人际关系模式（表里如一 vs 多面人格）
  - 心理健康（真实表达 vs 长期压抑）
  - 社交媒体行为（真实分享 vs 精心包装）

**高分角色**：华妃(0.9)、果郡王(0.9)、温太医(0.8)、甄嬛前期(0.7)
**低分角色**：皇后(0.2)、宁贵人(0.25)、甄嬛后期(0.3)

#### 8. 💡 求新倾向
- **测量**：拥抱变化、探索欲望 vs 偏好稳定熟悉
- **新增理由**：生活方式选择的决定因素
- **应用场景**：
  - 职业选择（创业/自由职业 vs 体制内稳定工作）
  - 生活方式（旅居/频繁换城市 vs 定居）
  - 风险承受能力

**高分角色**：果郡王(0.85)、华妃(0.7)、甄嬛后期(0.7)
**低分角色**：皇后(0.2)、沈眉庄(0.25)、温太医(0.3)

## 四、计分方式优化

### 4.1 从余弦相似度改为加权欧式距离

**原方法问题**：
```
余弦相似度只关注方向，不关注长度
例如：[0.5, 0.5, 0.5] 和 [1.0, 1.0, 1.0] 余弦相似度 = 1
但这两个人格明显不同（后者更极端）
```

**新方法优势**：
```
加权欧式距离考虑实际差距
距离 = √(Σ 权重ᵢ × (用户ᵢ - 角色ᵢ)²)
相似度 = 1 / (1 + 距离) × 100%
```

### 4.2 引入维度权重

不同维度对人格的影响程度不同：

| 维度 | 权重 | 理由 |
|-----|------|------|
| 策略思维 | 1.3 | 最能区分角色的核心特质 |
| 进取心 | 1.2 | 现代生活分化的决定因素 |
| 真实性 | 1.2 | 行为模式的根本差异 |
| 情感敏感 | 1.0 | 基础特质 |
| 外显能量 | 1.0 | 社交方式 |
| 求新倾向 | 1.0 | 生活选择 |
| 理想化 | 0.8 | 与情感敏感重合度较高 |
| 稳定自律 | 0.8 | 行为层面，影响相对较小 |

### 4.3 支持混合型人格

如果用户与多个角色都有较高相似度：
```
结果：你有65%的甄嬛后期特质，同时有52%的皇后特质
解读：你是战略型人格，但比纯粹的权谋者更灵活，愿意尝试新策略
```

## 七、示例：新旧模型对比

### 测试案例：都市白领小A

**背景**：
- 工作努力但不内卷
- 社交中会包装自己，但不虚伪
- 偏好稳定，不喜欢变化

#### 原6维测评结果
```
最相似角色：沈眉庄（65%）
理由：高自律、中策略、低外显能量
```

#### 新8维测评结果
```
最相似角色：浣碧（68%）
次要特质：沈眉庄（56%）

解读：
你是务实进取者（浣碧型）：
- 有明确的目标和行动力（进取心 0.7）
- 在工作中会适当包装自己（真实性 0.5）
- 偏好稳定的生活方式（求新 0.4）

同时有沈眉庄的坚韧和自律特质。
```

**对比**：新模型更准确地捕捉了"有野心但不激进"的特点。
