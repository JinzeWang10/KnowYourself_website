### 4.2 维度分数计算 (如适用)
```
四阶段混合校准算法：

阶段1: 维度得分累加 (Raw Score Accumulation)
- 情景选择题 (Q1-Q15): 每道题的选项包含 scores 对象，指定该选项对各维度的贡献值 (0-1)
- 例: { sincerity: 1.0 } 表示该选项对"真诚指数"维度贡献满分
- 累加所有题目中用户选择的选项对应的维度分数

阶段2: 锚点量表校准 (Anchor Calibration)
- 锚点题 (Q16-Q20): 5点Likert量表，每题对应一个维度
- 将1-5分转换为0-1范围: (答案 - 1) / 4
- 例: 选择"4 比较同意" → (4-1)/4 = 0.75

阶段3: 混合校准 (Hybrid Calibration)
- 计算每个维度的最大可能得分 (遍历所有情景题选项，找出每道题对该维度的最大贡献值并求和)
- 归一化情景题得分: relativeScore = rawScore / maxPossibleScore
- 混合公式: calibratedScore = α × relativeScore + β × anchorScore
  - α = 0.7 (情景选择题权重)
  - β = 0.3 (锚点量表权重)
- 最终得到每个维度的校准分数 (0-1范围)

阶段4: 欧氏距离角色匹配 (Euclidean Distance Matching)
- 预定义8个角色的五维度坐标 (0-1范围)
- 计算用户五维度分数与每个角色的欧氏距离:
  distance = sqrt(Σ(用户维度分 - 角色维度分)²)
- 归一化距离: normalizedDistance = distance / sqrt(5)
- 转换为相似度: similarity = 1 - normalizedDistance
- 按相似度排序，取前3名作为匹配结果
- 最终分数 = 第1名相似度 × 100
```
