SCL-90 开发输入模板

1. 基础信息
   1.1 量表标识
   • 量表 ID (英文，小写，用于代码): scl90
   • 量表标题 (中文): 症状自评量表
   • 量表标题 (英文): Symptom Checklist 90 (SCL-90)
   1.2 基本描述
   • 简短描述 (一句话，用于卡片展示): 全面评估心理症状严重程度，涵盖躯体化、强迫、抑郁、焦虑等 10 个症状维度
   • 测评目的 (详细说明，1-2 段): SCL-90 是世界上最著名的心理健康测试量表之一，包含 90 个项目，涵盖感觉、情感、思维、行为、人际关系以及生活习惯等广泛内容。它能够从感觉、情感、思维、意识、行为直至生活习惯、人际关系、饮食睡眠等多个角度，评定一个人是否有某种心理症状及其严重程度如何。

适用于 16 岁以上的成年人群，常用于心理咨询、临床诊断的初步筛查、疗效评估等场景。本量表可以反映受试者当前的心理症状状况，但不能作为诊断依据，需要专业人员结合其他信息进行综合判断。
• 测评分类:
◦ [x] 心理健康 (clinical)
◦ [ ] 人格测评 (personality)
◦ [ ] 职场生活 (career)
◦ [ ] 人际情感 (relationship)
◦ [ ] 其他: **\_\_\_**
• 预计完成时间: 15-30 分钟（一般在 20 分钟内完成）

   2.1 题目总览
   • 题目总数: 90,
   • 题目类型:
   ◦ [x] Likert 量表 (5 点),
   ◦ [ ] 单选题
   ◦ [ ] 多选题
   ◦ [ ] 其他: **\_\_\_**
   2.2 选项设置

   **指导语**：以下列出了有些人可能有的病痛或问题，请仔细阅读每一条，然后根据**最近一星期以内（包括今天）**您的实际感觉，选择最符合的答案。

   SCL-90 采用 5 级评分制：

   **📋 评分标准说明**（表格按评分从高到低排列，便于阅读）：

| 选项值 (Value) | 选项文本 (Text) |                  备注                  |
| :------------: | :-------------: | :------------------------------------: |
|       5        |      严重       |    自觉该症状的频度和强度都十分严重    |
|       4        |      偏重       |   自觉常有该项症状，其程度为中到严重   |
|       3        |      中等       |  自觉有该项症状，其严重程度为轻到中度  |
|       2        |      很轻       | 自觉有该项症状，但发生得不频繁、不严重 |
|       1        |      没有       |             自觉无该项症状             |

   **⚠️ 代码实现注意**：
   虽然上表按**评分从高到低**排列（便于阅读评分标准），但在代码实现时，选项数组必须按 **value 从小到大**排列（1→5），以符合 Likert 横向选项栏的视觉语义（左低右高）。详见下方"选项数组格式示例"。

2.2.1 题目字段规范 ⭐ 重要！

**每道题在代码实现时必须包含以下字段**：

| 字段名      | 必填 | 示例值                        | 说明                              |
| ----------- | ---- | ----------------------------- | --------------------------------- |
| `id`        | ✅   | `'scl90_1'`, `'scl90_2'`      | 题目唯一标识                      |
| **`type`**  | ✅   | **`'likert'`**                | **横向选项栏（Likert 量表专用）** |
| `question`  | ✅   | `'头痛'`                      | 题目文本                          |
| `dimension` | ✅   | `'somatization'`, `'anxiety'` | 所属维度（英文 ID）               |
| `options`   | ✅   | 见上方选项设置表格            | 选项数组（5 个选项）              |
| `required`  | ✅   | `true`                        | 是否必答（统一为 true）           |
| `reversed`  | ❌   | -                             | SCL-90 无反向题，不需要此字段     |

**⚠️ 关键注意事项**：

- ✅ **必须使用** `type: 'likert'` → 渲染为横向 Likert 量表选项栏
- ❌ **不要使用** `type: 'scale'` → 通用类型，可能渲染成垂直选项列表
- 📌 参考正确示例：[lib/scales/eq.ts](../lib/scales/eq.ts)（EQ 量表使用 `type: 'likert'`）
- 📌 错误示例：当前 SCL-90 使用了 `type: 'scale'`（需修复）

**选项数组格式示例**：

```typescript
options: [
  { value: 1, label: "没有" },
  { value: 2, label: "很轻" },
  { value: 3, label: "中等" },
  { value: 4, label: "偏重" },
  { value: 5, label: "严重" },
];
```

**🎨 选项顺序规范（极其重要！）**：

Likert 横向选项栏有**视觉语义**（符合心理学直觉和交通灯原则）：

- **左侧（绿色）** = 低分/消极/无症状/健康状态
- **右侧（红色）** = 高分/积极/有症状/需要关注

因此，选项数组**必须按照 value 从小到大排列**：

| 视觉位置 | value | label 语义           | 颜色提示 | 心理学意义       |
| -------- | ----- | -------------------- | -------- | ---------------- |
| 最左     | 1     | 没有/完全不同意/从未 | 🟢 绿色  | 健康/安全/低风险 |
| ←        | 2     | 很轻/不同意/很少     | 浅绿     | 轻微问题         |
| 中间     | 3     | 中等/中立/有时       | 🟡 黄色  | 中等程度         |
| →        | 4     | 偏重/同意/经常       | 橙色     | 需要关注         |
| 最右     | 5     | 严重/完全同意/总是   | 🔴 红色  | 高风险/需要帮助  |

**❌ 错误示例**（ANI 旧版本，已修复）：

```typescript
// 反了！高分在左，低分在右
{ value: 5, label: '非常频繁' },  // 左边（绿色）但语义是高分
{ value: 4, label: '经常' },
{ value: 3, label: '有时' },
{ value: 2, label: '很少' },
{ value: 1, label: '从未如此' },   // 右边（红色）但语义是低分
```

**✅ 正确示例**（SCL-90）：

```typescript
// 正确：低分在左，高分在右
{ value: 1, label: '没有' },       // 左边（绿色）且语义是低分 ✅
{ value: 2, label: '很轻' },
{ value: 3, label: '中等' },
{ value: 4, label: '偏重' },
{ value: 5, label: '严重' },       // 右边（红色）且语义是高分 ✅
```

**📐 验证方法**：
检查选项数组的第一项和最后一项：

- ✅ 第一项 `value` 最小，label 是"没有/不同意/从未"等消极语义
- ✅ 最后一项 `value` 最大，label 是"严重/同意/总是"等积极语义
- ❌ 如果反了，用户看到的视觉顺序会与语义相反，造成混淆！

4. 计分规则
   4.1 总分计算
   • 计分方法:
   ◦ [ ] 简单求和 (sum)
   ◦ [ ] 平均分 (average)
   ◦ [x] 自定义算法 (custom) - 计算总分、总均分、阳性项目数等
   • 分数范围 (单项分值 1-5 分):
   ◦ 最低分: 90 (90 x 1)
   ◦ 最高分: 450 (90 x 5)
   • 反向题处理规则: SCL-90 没有反向评分项目。
   4.2 维度分数计算
   • 因子分 (平均分) = 该维度内所有题目的得分总和 / 该维度内题目的数量。
   4.3 自定义计算逻辑
1. 总分 (Total Score)：90 个项目的得分总和,。
1. 总均分 (GSI, General Severity Index)：总分 ÷ 90。此指标衡量症状的强度和数量。
1. 阳性项目数 (PST, Positive Symptom Total)：得分大于 1（即得分 ≥2）的题目数量。
1. 阳性项目均分 (PSDI, Positive Symptom Distress Index)：总分 ÷ 阳性项目数。此指标衡量受试者痛苦程度。

5. 结果解读

   5.1 筛查阳性标准

   符合以下**任一条件**即为筛查阳性，提示可能存在心理健康问题，建议寻求专业评估：

   1. **总分 > 160**
   2. **任一因子分（平均分）≥ 2**
   3. **阳性项目数 > 43**

   5.2 总分/总均分解读

   **分数段 1：心理健康（低风险）**

   - **总分范围**：90 - 160
   - **总均分范围**：1.0 - 1.77
   - **颜色代码**：`#10b981`（绿色）
   - **简短描述**：症状水平低于临床筛查界限，心理状态总体健康
   - **详细描述**：

     您的 SCL-90 测评结果显示，您当前的心理症状水平处于正常范围内。这表明您在最近一周内，各方面的心理症状都较轻微，不会对日常生活、工作和人际关系造成明显影响。

     虽然偶尔可能会体验到一些不适感或情绪波动，但这些都是生活中的正常现象。保持良好的生活习惯，适当关注自己的情绪变化，继续保持积极的心态。

     如果未来感到压力增大或出现持续的不适感，可以考虑寻求心理咨询的支持。

   - **建议**：
     - 保持规律的作息和健康的生活方式
     - 培养积极的兴趣爱好，丰富精神生活
     - 维护良好的人际关系，建立支持系统
     - 学习压力管理技巧，提升心理韧性
     - 定期进行自我反思和心理健康评估

   **分数段 2：筛查阳性（需关注）**

   - **总分范围**：161 - 450
   - **总均分范围**：1.78 - 5.0
   - **颜色代码**：`#ef4444`（红色）

     5.3 因子分（维度）解读标准

   每个因子的平均分计算方式：该因子所有题目得分之和 ÷ 该因子题目数量

   **因子分评估标准**：

   - **< 2.0**：该维度症状在正常范围内
   - **≥ 2.0**：该维度存在阳性症状，需要重点关注
   - **≥ 3.0**：该维度症状较为严重，强烈建议专业评估
   - **≥ 4.0**：该维度症状非常严重，需要立即寻求专业帮助

   **各因子解读**：

   1. **躯体化（F1）**：反映主观的身体不适感。高分提示可能存在心身问题，也可能是焦虑、抑郁等心理问题的躯体化表现。

   2. **强迫症状（F2）**：反映明显的强迫思维和强迫行为。高分提示存在难以摆脱的想法或重复性行为，影响日常效率。

   3. **人际关系敏感（F3）**：反映人际交往中的不适感和自卑感。高分提示在与他人互动时容易感到紧张、被误解或不被接纳。

   4. **抑郁（F4）**：反映抑郁情绪和相关症状。高分提示情绪低落、兴趣减退、缺乏活力等，需要特别关注。

   5. **焦虑（F5）**：反映焦虑情绪和相关躯体表现。高分提示存在紧张不安、担忧、惊恐等症状。

   6. **敌对（F6）**：反映攻击性思维、情感和行为。高分提示容易愤怒、与人争执或有攻击冲动。

   7. **恐怖（F7）**：反映恐惧反应。高分提示对特定场所、情境或社交场合存在明显恐惧。

   8. **偏执（F8）**：反映偏执性思维。高分提示存在猜疑、不信任、认为别人对自己有敌意等想法。

   9. **精神病性（F9）**：反映精神分裂症样症状。高分提示可能存在思维、知觉或行为方面的异常，需要尽快专业评估。

   10. **其他（F10）**：主要反映睡眠和饮食问题。高分提示存在睡眠障碍或饮食问题。

   5.4 特殊项目警示

   - **题目 15（想结束自己的生命）**：如果此题得分 ≥ 3，提示存在自杀风险，需要立即关注
   - **题目 59（想到死亡的事）**：如果此题得分 ≥ 3，提示存在消极想法，需要重点评估
   - **题目 63（有想打人或伤害他人的冲动）**：如果此题得分 ≥ 4，提示存在攻击风险，需要立即关注

   - 北京心理危机研究与干预中心：010-82951332
   - 或拨打当地精神卫生中心 24 小时热线
